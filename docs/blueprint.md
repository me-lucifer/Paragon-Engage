# **App Name**: Paragon Engage

## Core Features:

- Dashboard: Display key performance indicators (KPIs) and metrics in an easily digestible format.
- Lead Management: Allow users to create, edit, and manage leads.
- Engagement Tracking: Track user engagement across different channels.
- Content Recommendations: Leverage AI to recommend relevant content to users based on their profiles and behavior, using a recommendation tool.
- Reporting: Generate reports on key engagement metrics.
- User Management: Admin users can manage all users with proper permission levels and data access.
- Responsive Layout: Ensure the application is responsive and adapts to different screen sizes.

## Style Guidelines:

- Primary color: #29ABE2 (light blue) for a professional and trustworthy feel.
- Accent color: #FF9800 (orange) for call-to-action buttons to draw attention.
- Background color: #F5F5F5 (light gray) for a clean and modern look.
- Text color: #1F2937 (dark gray) for primary text, #4B5563 (gray) for secondary text to ensure readability.
- Font: 'Inter' (sans-serif) for a clean and modern look; sizes 14px, 16px, 20px, 28px.
- Modern B2B SaaS layout: Top navbar + left sidebar + main content area.
- Surface/Panel: #FFFFFF (white) with subtle shadow and 12px radius for a layered effect.
- Subtle hover states on buttons and interactive elements to provide visual feedback.
- Outline icons for a clean and modern look.
